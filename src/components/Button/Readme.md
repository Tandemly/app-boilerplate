A Basic button:

```js
<Button text={'Click Me'} />
```

A primary color button:

```js
<div className="button-example">
  <Button text={'Primary'} className="is-primary" />
  <Button text={'Secondary'} className="is-secondary" />
  <Button text={'Accent'} className="is-accent" />
</div>
```

```js
<div className="button-example">
```
